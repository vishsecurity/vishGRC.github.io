# GRC Suite - Quick Start Guide

## ‚ö° 5-Minute Setup

### Step 1: Login (30 seconds)
```
Username: admin
Password: admin123
```

### Step 2: Change Admin Password (1 minute)
1. Go to **User Management**
2. Click edit on admin user
3. Enter new password
4. Click Save

### Step 3: Configure Settings (2 minutes)
1. Go to **Settings**
2. Update Company Name
3. Upload logos (optional)
4. Set brand colors (optional)
5. Configure AI if needed
6. Click **Save Settings**

### Step 4: Load Framework (1 minute)
1. Go to **Compliance**
2. Click **Load Framework**
3. Select framework (e.g., ISO 27001)
4. Click **Load Framework**

### Step 5: Start Working! (30 seconds)
- View Dashboard for overview
- Add vendor assessments
- Update compliance status
- Create VAPT reports
- Track privacy records

## üìö Common Tasks

### Add a New User
```
1. User Management ‚Üí Add User
2. Fill username, email, password
3. Select role (Admin/Auditor/Analyst/Viewer)
4. Set permissions
5. Click Save
```

### Create Vendor Assessment
```
1. Vendor Risk ‚Üí New Assessment
2. Fill questionnaire sections:
   - Company Information
   - Information Security
   - Data Privacy
   - Business Continuity
3. Submit Assessment
4. Review auto-calculated risk score
5. Change status as needed
```

### Update Compliance Status
```
1. Compliance ‚Üí Select Framework
2. Click on status badge
3. Select new status
4. Add evidence in Evidence column
5. Add notes in Notes column
6. Changes auto-save
```

### Create VAPT Report
```
1. VAPT Reporting ‚Üí New Report
2. Update report title and client name
3. Click Add Finding
4. Fill finding details:
   - Title
   - Severity (Critical/High/Medium/Low)
   - CVSS Score
   - Description
   - Evidence
   - Remediation
5. Use AI Generate for remediation
6. Add executive summary
7. Export to PDF
```

### Track Privacy Record
```
1. Privacy Management ‚Üí Select tab
   - ROPA for processing activities
   - PII for personal data inventory
   - DPIA for impact assessments
   - Consent for consent tracking
   - DSAR for access requests
2. Click Add Record
3. Fill required fields
4. Create Record
```

### ISO 27001 Compliance Workflow
```
Day 1: Load ISO 27001 framework
Day 2-7: Review and update control status
Day 8-14: Add evidence for compliant controls
Day 15-21: Fix non-compliant controls
Day 22-28: Generate AI summary
Day 29-30: Export PDF report
```

### Vendor Onboarding Workflow
```
1. Create vendor assessment
2. Send questionnaire link
3. Vendor submits responses
4. Review risk score
5. Approve/Reject vendor
6. Export vendor list
```

### VAPT Project Workflow
```
Week 1: Create report, start testing
Week 2: Document findings with evidence
Week 3: Add remediation steps (use AI!)
Week 4: Generate executive summary
Week 5: Export final PDF report
```

### Privacy Compliance Workflow
```
Month 1:
- Document ROPA (processing activities)
- Create PII inventory

Month 2:
- Conduct DPIA for high-risk processing
- Track consent for marketing

Month 3:
- Handle DSAR requests
- Review and update records
```

## üîß Troubleshooting

- Close unused tabs
- Export old data
- Use Chrome/Firefox
